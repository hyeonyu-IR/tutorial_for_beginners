# Installing Python (Windows/macOS/Linux)

## Windows
- Recommended: **Python.org installer** (3.10+). During install, check **“Add Python to PATH.”**
- Alternative: **Miniforge/Conda** for isolated environments.

## macOS
- Recommended: **Miniforge** (ARM/x86 friendly) or **Homebrew** `python@3`.

## Linux
- Use system package manager for Python 3, plus `pip`.

After install, verify:
```bash
python --version
pip --version
```
