# Python for R Users (Fast Map)

| Concept | R | Python |
|---|---|---|
| Assign | `x <- 5` | `x = 5` |
| Vector/Array | `c(1,2,3)` | `list` or `numpy.array` |
| Data frame | `data.frame()` | `pandas.DataFrame()` |
| Indexing | 1‑based | 0‑based |
| Package | `install.packages()` | `pip install` |

Idioms:
```python
# list comprehension
squares = [i*i for i in range(10)]

# dict (named list)
patient = {"id": 123, "age": 67, "sex": "F"}
```
